# Upscale Pixel Art                                                                              
   
  Upscale pixel art from 1x to 16x directly in your browser                      
                  
  ## Features                                                                                           
                  
  - Crisp, blocky output using pixelated rendering
  - Compare with original by hovering over the canvas
  - Download upscaled image
                                                                                                        
  ## Usage
                                                                                                        
  Open [upscalepixel.art](upscalepixel.art) or download and open `index.html` directly in your browser                                                                  
   
  ## How to Use                                                                                         
                  
  1. **Choose File** to upload a pixel art image
  2. Use the slider to select the upscale factor (1x–16x, default: 4x)                                    
  3. **Download** to save the upscaled image
