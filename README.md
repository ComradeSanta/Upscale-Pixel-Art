# Upscale Pixel Art                                                                              
   
  Upscale pixel art from 1x to 15x, processed only in your browser, no uploading to any server                      
                                                                                                        
  ## Usage
                                                                                                        
  Open [upscalepixel.art](upscalepixel.art) or download `index.html` and open it directly in your browser                                                                  
   
  ## How to Use                                                                                         
                  
  1. **Choose File** to upload a pixel art image
  2. Use the slider to select the upscale factor (1x–15x, default: 4x)                                    
  3. **Download** to save the upscaled image
